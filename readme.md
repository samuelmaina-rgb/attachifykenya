# Attachify Kenya

Attachify Kenya is a front-end web platform designed to connect university and college students with attachment opportunities across Kenya. The website provides an easy way for students to explore opportunities in different industries while allowing companies to register and advertise available attachment positions.

## Features

- Responsive homepage with a modern user interface
- Category pages for:
  - Technology
  - Medicine
  - Architecture
- Student and company login system (front-end)
- Student profile registration page
- Company registration page
- About Us page
- Contact Us page
- Consistent navigation and footer across all pages
- Professional card layouts with images and hover effects

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Poppins)




## Website Pages

- **Home** – Introduces Attachify Kenya and showcases available industries.
- **Technology** – Technology attachment opportunities.
- **Medicine** – Medical attachment opportunities.
- **Architecture** – Architecture attachment opportunities.
- **About** – Information about the platform, its mission and vision.
- **Contact** – Contact information and enquiry form.
- **Login** – Allows users to choose between student and company login.
- **Student Registration** – Create a student profile.
- **Company Registration** – Register a company account.
